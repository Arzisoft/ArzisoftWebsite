# Arzisoft Website

Official website for **Arzisoft** — a solution development company that builds custom software, web platforms, and digital products for businesses.

## Project Structure

```
├── frontend/
│   ├── public/assets/       # Images, fonts, icons
│   └── src/
│       ├── components/      # Reusable UI components
│       ├── pages/           # Page templates
│       ├── styles/          # CSS/stylesheets
│       └── scripts/         # Client-side JavaScript
├── backend/
│   ├── node/
│   │   ├── src/
│   │   │   ├── routes/      # API routes
│   │   │   ├── controllers/ # Request handlers
│   │   │   ├── middleware/  # Auth, logging, etc.
│   │   │   └── models/      # Data models
│   │   └── config/          # Environment config
│   └── python/
│       └── src/
│           ├── services/    # Business logic services
│           └── utils/       # Utility scripts
├── docs/                    # Documentation
└── index.html               # Coming soon page (live)
```

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Python
- **Design System:** Figma — [Arzisoft Website Design System](https://www.figma.com/design/0EMxoUCgwEUALKX0KBxjo6/Arzisoft---Wesbsite-Design-System)
